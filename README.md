# formulario_completo
Formulário completo 
Uma atividade aplicada na faculdade para desenvolver um formulário com: 
.HTML
.CSS 
. JavaScript
A proposta principal foi aprimorar as habilidades em validação de formulários, implementando verificações de campos obrigatórios, formatos corretos de entrada e feedback visual para o usuário.

O foco esteve voltado para o uso do JavaScript na validação dos dados antes do envio, garantindo que todas as informações fossem preenchidas corretamente e atendendo aos critérios estabelecidos. Além disso, o projeto também reforçou o uso de boas práticas de estruturação (HTML) e estilização (CSS) para uma melhor experiência de usuário.
